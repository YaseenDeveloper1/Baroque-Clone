# 🛍 Baroque Clone – Responsive Fashion eCommerce UI

A fully responsive eCommerce front-end project inspired by a modern fashion brand website.  
This project focuses on clean layout structure, interactive UI behavior, and responsive design principles.

🔗 **Live Demo:**  
https://baroque-clone-vert.vercel.app/

---

## 🚀 Project Overview

Baroque Clone replicates the front-end experience of a modern fashion eCommerce store, including product listings, category pages, and detailed product views.

The goal of this project was to practice:

- Multi-page layout structuring
- Responsive UI design
- Interactive product behavior
- Clean visual hierarchy

---

## ✨ Features

- 🛒 Multi-page eCommerce layout
- 📱 Fully responsive (Desktop / Tablet / Mobile)
- 🖼 Interactive product image zoom on hover
- 🧭 Navigation bar & dropdown menus
- 🗂 Category & product listing pages
- 🧾 Product detail view layout
- ⚡ Smooth UI interactions

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Lightweight jQuery plugin (for image zoom interaction)
